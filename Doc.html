What assets/pages/ does:

All functions are declared here:

  • Shows pages on index.php.

  • Registers the user after checking:

    • If user has filled every field.
    • Password and confirm password.

    Then stores the password after encryting using bf (BlowFish) algo.

    •   $form_data -> username, name, email, password, cpassword

  • For login - to check if user is already present or not

    • Login possible using email or username
    • Password is checked using password_verify func
    • User status is changed to active[active->true (fieldname in db)]

      $form_data -> username_email, password

What assets/pages/ does:

  • If request sent from signup it sends request to assets/pages/functions.php for registering user(also validates user).
